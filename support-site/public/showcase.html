The idea

Planning apps make you do the planning.

Every to-do app hands you an empty grid and makes you figure out when to do everything, drag blocks around, and remember to leave on time. People don't think in calendars — they think in quick fragments: a thought on the walk, a reminder fired off mid-meeting.

Planfect's bet: you should be able to say one casual sentence — even a whole afternoon at once, by voice — and get a realistic, scheduled day back in seconds. The app does the when: it fits things around your work and sleep, adds real travel time so you arrive on time, learns your habits, and only asks when it's genuinely unsure.

The product

How it's built

Architecture

A native iOS client, a serverless TypeScript backend, and an LLM agent that doesn't just chat — it reasons about your schedule, calls tools, and commits real database writes.

// the planning round-trip iOS app (SwiftUI) ──HTTPS──▶ /plan Supabase Edge Function (Deno · TypeScript) │ ├─▶ AI agent loop (OpenAI, tool-calling) │ tools: get_schedule · schedule_tasks · │ estimate_commute · web_search · │ set_routine · remember_preference … │ ├─▶ Google Maps (Routes + Geocoding — real travel time) │ └─▶ Postgres (Row-Level Security) tasks · time_blocks · routines · preferences

iOS · SwiftUI

Chat UI w/ markdown, day/week/month timetable, Swift Charts insights, WidgetKit (App Groups), EventKit, Speech, Sign in with Apple, local notifications.

Backend · Supabase

Postgres + RLS, Edge Functions in Deno/TypeScript, auth, migrations. A timezone-aware scheduling engine (pure, unit-tested) places tasks into free slots.

AI · agent loop

Multi-step tool-calling over OpenAI: reads context, proposes, confirms, then writes. Pluggable provider/model; usage & cost logged for analytics.

Engineering highlights

The hard parts

🌍 Timezone-correct scheduling

The planner schedules in the user's local time. It looks up cross-timezone events (a London match watched from the US) and converts to the user's clock. Fixed a class of bugs where “today” drifted to a past date in long, multi-day chats — by stamping the authoritative date onto the latest turn for recency.

🧭 Product-minded placement

Commute is scheduled before a pinned task (you leave early, arrive on time — not the activity pushed later). A voice brain-dump of several tasks is planned in a single batch; genuine conflicts are surfaced, never silently mis-scheduled.

⚡ Latency, halved

Cut planner latency ~60% by short-circuiting a redundant LLM round-trip once tasks land on the calendar, skipping needless schedule lookups (the calendar is already in context), and parsing dates once at decode.

🔒 Shipping rigor

RLS-secured data, App Groups for the widget, distribution signing → TestFlight → App Store pipeline (automated via API key), a password-gated analytics dashboard, CI, and prompt-level content guardrails.
